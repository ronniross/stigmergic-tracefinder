# stigmergic-tracefinder
A series of scraping pipelines that collect data and create references for authors and works. It maps hidden networks of influence, tracing how concepts evolve and propagate across time and disciplines.

| Database / API | Focus Area | Python Script (`.py`) | Jupyter Notebook (`.ipynb`) |
| :--- | :--- | :--- | :--- |
| **arXiv** | Physics, Math, Comp-Sci, & Quantitative Biology | `arxiv-stigmergic-trace-finder.py` | `arxiv-stigmergic-trace-finder.ipynb` |
| **bioRxiv / medRxiv** | Life Sciences, Zoology, & Biology Preprints | `extended-trace-finder.py` | `extended-trace-finder.ipynb` |
| **Europe PMC** | Peer-Reviewed Life Sciences & Biomedical | `extended-trace-finder.py` | `extended-trace-finder.ipynb` |
| **OpenAlex** | Massive Global Open-Source Aggregator | `extended-trace-finder.py` | `extended-trace-finder.ipynb` |
