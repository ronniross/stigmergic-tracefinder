# stigmergic-tracefinder
A series of scraping pipelines that collect data and create references for authors and works. It maps hidden networks of influence, tracing how concepts evolve and propagate across time and disciplines.

A departure from hierarchical, top-down knowledge graphs. This module creates non-linear, subterranean semantic mappings, allowing the language model to connect seemingly disparate fields (e.g., indigenous botany and quantum mechanics) through lateral, highly resilient conceptual roots.
